Developers

There is no API key - Here is the list of endpoints.

  • /getUsers (Data Endpoint)
  • /random-message (Data Endpoint)
  • /random-message?limit=2 (Data Endpoint w/ Parameter)
  • /write (POST Endpoint)
  • /get-music-list (Data Endpoint)
  • /get-music-list?limit=3 (Data Endpoint w/ Parameter
  • /suggest (POST Endpoint)

The default data format is JSON. If you send an Accept header of "text/xml", the data will be returned in XML format.

The default request method is GET. If you send a HEAD request, only the HTTP headers will be returned.
